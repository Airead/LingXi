搜索，万物

模块化的 macOS 搜索启动器，支持 Lua 插件 — 应用、文件、书签、剪贴板、片段，一键直达。

⬇ 下载 ★ GitHub
最新版本

功能特性

一个启动器，多种数据源

🔌

Lua 插件系统

使用沙箱化 Lua 插件扩展灵犀。注册命令、构建 Webview 界面，无需重新编译即可发布功能。

💻

应用与文件搜索

模糊匹配已安装应用和 Spotlight 文件。输入 f 搜索文件，fd 搜索文件夹 — 回车打开，或在 Finder 中显示。

📋

剪贴板历史

使用 cb 前缀搜索剪贴板历史。容量受限，可即时粘贴回上一个聚焦应用。

⚡

片段与自动展开

保存文本片段，输入触发词即可在任意位置展开。使用 sn 搜索，或开启自动展开自动完成。

⚙

系统设置与书签

直接跳转任意系统设置面板。使用 bm 搜索浏览器书签 — 支持 Safari、Chrome 等。

🛠

原生 macOS

基于 SwiftUI 与 Swift 6.2 严格并发构建。默认 MainActor 隔离，原生面板，深度系统集成。

快捷键

熟悉的快捷键，完全可配置

呼出 / 隐藏面板 Option + Space
打开选中项 Return
选择第 1–9 项 Cmd + 1…9
上下导航 ↑ / ↓
自动补全 Tab
删除条目 Cmd + Delete
关闭面板 Esc
替代动作 Cmd / Option + Return
