Search, Anything

A modular macOS search launcher with Lua plugin support — apps, files, bookmarks, clipboard, snippets, all in one keystroke.

Latest release

Features

One launcher, many providers

🔌

Lua Plugin System

Extend LingXi with sandboxed Lua plugins. Register commands, build webview UIs, and ship features without recompiling the app.

💻

App & File Search

Fuzzy-match installed apps and Spotlight files. Type f for files, fd for folders — open with Enter or reveal in Finder.

📋

Clipboard History

Searchable clipboard history with cb prefix. Capacity-bounded, instantly pasted back into the previously focused app.

⚡

Snippets & Auto-Expand

Save text snippets, then expand them by typing a trigger anywhere. Search with sn or let auto-expand do the work.

⚙

System Settings & Bookmarks

Jump straight into any System Settings pane. Search browser bookmarks with bm — Safari, Chrome, and more.

🛠

Native macOS

Built with SwiftUI and Swift 6.2 strict concurrency. MainActor-isolated by default, native panel, and tight system integration.
