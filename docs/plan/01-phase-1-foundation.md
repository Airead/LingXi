# 阶段一：基础架构 — 分步计划

## 目标

完成应用的基础骨架：菜单栏常驻、全局快捷键呼出、浮动搜索面板显示/隐藏。

---

## Step 1：菜单栏应用 + 隐藏 Dock 图标

**改动：**
- 添加 `Info.plist`，设置 `LSUIElement = true`
- 修改 `LingXiApp.swift`，使用 `MenuBarExtra` 替代 `WindowGroup`
- 菜单栏图标使用系统图标 `magnifyingglass`
- 菜单中包含"退出 LingXi"选项

**验证：**
- [ ] 运行后 Dock 中无图标
- [ ] 菜单栏出现放大镜图标
- [ ] 点击图标弹出菜单，点"退出 LingXi"可关闭应用

---

## Step 2：全局快捷键

**改动：**
- 新建 `HotKey/` 模块，封装全局快捷键注册逻辑
- 使用 `NSEvent.addGlobalMonitorForEvents` + `addLocalMonitorForEvents` 监听 `⌥ Space`
- 按下快捷键时在控制台打印日志（后续替换为显示面板）

**验证：**
- [ ] 应用运行后，在任意界面按 `⌥ Space`
- [ ] Xcode 控制台输出快捷键触发日志
- [ ] 重复按下可重复触发

---

## Step 3：浮动搜索面板（空面板）

**改动：**
- 新建 `Panel/` 模块，创建 `FloatingPanel`（继承 `NSPanel`）
- 面板特性：无标题栏、居中显示、浮动在最上层、圆角
- 新建 `PanelManager` 管理面板的显示/隐藏
- 面板内暂时只显示一个占位文本"LingXi"

**验证：**
- [ ] 调用显示方法后，屏幕中央出现一个无标题栏的浮动面板
- [ ] 面板显示占位文本"LingXi"
- [ ] 面板始终浮在其他窗口之上

---

## Step 4：串联 — 快捷键控制面板显示/隐藏

**改动：**
- 将 Step 2 的快捷键回调连接到 Step 3 的 `PanelManager`
- 实现 toggle 逻辑：按快捷键切换面板显示/隐藏
- 点击面板外部区域自动隐藏面板
- 按 `Esc` 隐藏面板

**验证：**
- [ ] 按 `⌥ Space` 面板出现在屏幕中央
- [ ] 再按 `⌥ Space` 面板消失
- [ ] 面板显示时，点击桌面其他区域，面板自动隐藏
- [ ] 面板显示时，按 `Esc` 面板隐藏
- [ ] 菜单栏图标点击菜单仍正常工作

---

## 完成标准

阶段一完成后，应用表现为：
1. 启动后仅在菜单栏显示图标
2. 按 `⌥ Space` 弹出一个空的浮动面板
3. 再次按快捷键、按 Esc、或点击外部区域可隐藏面板
4. 菜单栏可退出应用
