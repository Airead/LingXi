# Emoji Search Plugin for LingXi

Search and paste emoji via the launcher.

## Usage

Type `e` followed by a keyword to search for emojis.

### Examples

- `e` — Show all emoji groups
- `e cat` — Search for cat-related emojis
- `e @animals` — Show all emojis in the animals group
- `e face @smileys` — Search for "face" within the smileys group

### Actions

- **Enter** — Paste the selected emoji into the previous application
- **Cmd+Enter** — Copy the emoji to clipboard

## Data

Emoji data is sourced from the Unicode CLDR and provided by the WenZi project.
